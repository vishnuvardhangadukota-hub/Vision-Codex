## 🌟 Overview

**Vision Codex** is a real-time AI-powered platform that detects bias and fairness issues in datasets.

It helps developers, researchers, and organizations ensure ethical AI by analyzing CSV datasets and providing:

- 📊 Bias detection
- 🧠 AI-generated explanations
- ⚠️ Risk classification (Safe / Warning / Critical)
- 📈 Insightful dashboard visualization

## 🧪 How It Works

1. Upload a CSV dataset  
2. System analyzes distribution patterns  
3. Detects imbalance & bias  
4. Generates AI-based explanation  
5. Displays insights in dashboard
